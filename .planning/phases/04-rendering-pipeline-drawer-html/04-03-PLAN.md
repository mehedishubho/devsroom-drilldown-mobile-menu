/* ========================================================================
   Phase 4: Drawer / Panel / Overlay Base Layout

                visibility var( --ddmm-transition-duration ) ease;

- Use `var( --ddmm-* )` for all themeable values so Phase 6 Style Tab can override them.
- The `.ddmm-is-open` block anticipates Phase 5 (D-26 recommendation: toggle on `.ddmm-widget`). Define it now so the transition is ready.
- Do NOT add `content_template()` styles or editor-iframe-specific hacks — the `.ddmm-editor-preview` class handles the editor preview styling.
