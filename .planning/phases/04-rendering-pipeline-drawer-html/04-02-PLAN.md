| Elementor settings → PHP widget | Widget settings (brand_source, brand_image, brand_text, nav_label, show_back_title) are admin-configured; brand_image URL and brand_text are free-text inputs that cross into HTML via DrawerRenderer. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
