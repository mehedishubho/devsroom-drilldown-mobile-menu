**Algorithm (stack-based, per D-03, D-09, ARCHITECTURE.md Pattern 5):**

1. If `$items` is empty, return `[]` immediately.
2. Initialize `$tree = []`, `$stack = []`, `$id = 0`.
3. For each `$item` in `$items`:
   a. Increment `$id` (sequential from 1, per D-10).
   b. Cast `$depth = (int) ($item['depth'] ?? 0)`.
   c. Extract `$title = $item['label'] ?? ''`.
   d. **Skip if `$title` is empty** (Pitfall 5: prevents phantom menu items). Use `continue`.
   e. Extract URL from `$item['url']`:
      - If `is_array($item['url'])` and `!empty($item['url']['url'])`, use `$item['url']['url']`.
      - Else if `is_string($item['url'])` and non-empty, use it directly.
      - Otherwise `$url = ''`. (Per Pitfall 1: URL control returns array.)
   f. Set `$target = (!empty($item['new_tab']) && 'yes' === $item['new_tab']) ? '_blank' : ''`. (Per Pitfall 2: SWITCHER returns 'yes'/'' not boolean.)
   g. Build the node array with the exact 7 fields per D-02:
      ```php
      $node = [
          'id'           => $id,
          'title'        => $title,
          'url'          => $url,
          'target'       => $target,
          'classes'      => [],
          'has_children' => false,
          'children'     => [],
      ];
      ```
   h. **Auto-clamp stack** (per D-03): `while (count($stack) > $depth) { array_pop($stack); }`
   i. If `$depth === 0 || empty($stack)`:
      - Push `$node` to `$tree`: `$tree[] = $node;`
      - Reset stack to reference the new root: `$stack = [ &$tree[count($tree) - 1] ];`
   j. Else (child item):
      - Get parent reference: `$parent = &$stack[count($stack) - 1];`
      - Attach to parent: `$parent['children'][] = $node;`
      - Set has_children: `$parent['has_children'] = true;`
      - Push child onto stack: `$stack[] = &$parent['children'][count($parent['children']) - 1];`
   k. `unset($node);` (per Pitfall 4: break reference to avoid next-iteration interference)
4. Return `$tree`.

**File header:** Use the same PHPDoc pattern as WpNavTree.php: `@package Devsroom_DDMM\MenuBuilder`, class-level docblock explaining "Converts flat repeater items with depth field into nested tree using stack-based algorithm. Pure data — no HTML, no Elementor dependency."

**Important implementation details:**
- The `while (count($stack) > $depth)` auto-clamp: when depth jumps from 0 to 3, stack has 1 element, 1 > 3 is false, so nothing pops. The item then hits the `else` branch and attaches as a child of the stack top (the root). Effective depth becomes 1 (clamped from 3).
- The `&` reference operators are critical: `$stack = [ &$tree[count($tree) - 1] ]` and `$stack[] = &$parent['children'][...]`. Without references, modifications to child arrays would not propagate back into the tree.
- Do NOT import any Elementor classes. This file is pure PHP.
